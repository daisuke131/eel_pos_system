# eel_pos_system

## 起動手順

### 仮想環境作成

```
$ python3 -m venv .venv
```

### 立ち上げ

#### Mac

```
$ source .venv/bin/activate
```

#### windows

```
$ source .venv/Scripts/Activate
```

### 使用ライブラリ一括インストール

```
$ pip install -r requirements.txt
```

### 起動

```
$ python run.py
```

### exe化

```
$ python -m eel run.py web --onefile --noconsole
```
